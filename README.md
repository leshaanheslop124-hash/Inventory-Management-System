# Inventory Management System

A desktop-based inventory management system built using C# Windows Forms.

## Overview

This system allows users to manage inventory items, suppliers, and sales while maintaining accurate stock control through database integration.

The project focuses primarily on stability, reliability, and structured code design rather than advanced UI styling, because reliablity is my main goal.


## Features

- Add, edit, and delete inventory items
- Supplier management
- Sales tracking
- Stock quantity updates
- Input validation and error handling
- SQL database integration

## Technologies Used

- C#
- Windows Forms
- SQL Server
- ADO.NET

## How to Run

1. Clone the repository
2. Open the solution in Visual Studio
3. Update the database connection string
4. Run the project
